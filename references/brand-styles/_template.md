# 客户 Brand Style 卡片 · 模板(v5-beta 新增)

> **用法**:每次给新客户做培训前,**复制这个模板**到 `brand-styles/{客户简称}-brand.md`,填一份。
> AI 生成 PPT 时会**自动读取这份卡片**,自动应用客户色 / 字体偏好 / 合规约束。
>
> **借鉴来源**:power-design 的 brand-style.md 概念,简化为饶秋培训交付场景需要的 4 维度。
> **跟"客户色"机制的关系**:这是客户色 v2.0 — 不只是 hex 值,还包括字体偏好 / Tone / 合规禁用。

### Tone · 语言风格

| 项 | 值 |
|---|---|
| 行业语言风格 | 正式 / 半正式 / 口语化 / 行业黑话 |
| 学员习惯称呼 | "您 / 您们" / "你 / 你们" / "同学们" / "各位" |
| 数据陈述偏好 | 量化数字(40%↑)/ 形容词(显著提升)/ 案例叙事 |
| 比喻 / 类比 | 鼓励 / 适度 / 不要 |

**饶秋默认**:"你 / 你们" 直白 + 大白话 + 量化数字优先,不"赋能 / 抓手 / 链路 / 底层逻辑"。
**特殊情况**:有些金融 / 国企客户要求"您"和正式用语,这里填具体要求即可。

## 怎么用这份卡片

**AI 自动调用**:饶秋 SKILL.md Step 1 第 4 问"客户色 / 主色偏好"扩展成"Brand Style 卡片":
1. 用户提供客户简称
2. AI 在 `references/brand-styles/` 找 `{简称}-brand.md`
3. 没找到:用户填一份(或回答 6 问澄清扩展版)
4. 有找到:**自动应用**(客户色 / Tone / 合规约束)

**手动维护**:
- 每个新客户:复制 `_template.md` → `{简称}-brand.md` → 填空
- 已有客户更新:每次培训后回填 Tracking 表
- 客户 Tone 变化:直接改卡片,下次自动用新版

**版本控制**:
- `_template.md` 是模板,不要直接改这个
- 每个客户单独一份卡片
- 改完直接生效,不需要重启 skill
